# Data folder

This folder is intentionally kept almost empty.

The original and intermediate CSV files are not included in the repository because they may be large or contain data that should not be published.

Expected local files for full notebook execution:

- `0_Kirill_Viksman.csv` — original dataset;
- `1_Kirill_Viksman.csv` — engineered dataset created after feature engineering;
- `2_Kirill_Viksman.csv` — output from the linear-model stage;
- `3_Kirill_Viksman.csv` — output from the nonlinear-model stage;
- `4_Kirill_Viksman.csv` — final output from the feature-selection stage.

The final public-safe output included in this repository is stored as:

- `outputs/final_submission.csv`
