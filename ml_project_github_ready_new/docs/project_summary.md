# Project summary

This project solves two machine learning tasks in one pipeline:

1. **Price prediction** — regression problem.
2. **Churn probability prediction** — classification problem.

The notebooks cover data preprocessing, model training, model comparison, feature selection and preparation of the final submission file.

## Portfolio positioning

This repository can be used as a portfolio ML project because it shows:

- practical feature engineering;
- comparison of several ML model families;
- regression and classification in one project;
- feature selection and model simplification;
- final reproducible submission structure.

End-to-end machine learning project: feature engineering, regression, classification, model selection and feature selection.
